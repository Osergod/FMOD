Activitat FMOD  
Enunciat:  
Prepareu una petit projecte de FMOD amb: - 
Una event de so Timeline amb un asset d’una cançó fent loop des de FMOD. - - - - - 
Un event de so amb un multiinstrument que utilitzi dos assets generats amb 
jfxr.com 
L’event de so ha de tenir una modulació de Random aplicada sobre el pitch, 
de tal manera que cada cop que soni ho faci amb un pitch lleugerament 
diferent. 
Un fader group de SFX i un fader group de Music, ambdòs enrutats als 
corresponents VCA de SFX i de Music enrutats a un altre VCA per volum 
General. 
Dos paràmetres de tipus continuous. 
L’event de Timeline de la cançò ha de tenir dos efectes de so, els quals es 
vegin modificats en base als valors dels dos paràmetres. 
Prepareu un petit projecte de Unity, vinculat amb l’FMOD, de tal forma que mostri: - - - - 
Botó per fer pausa/resume de la cançó. 
Botó per fer sonar el so  
Sliders que modifiquin el valor dels paràmetres dels efectes de FMOD.  
El fet de pulsar escape faci una “pausa”. - 
Durant la pausa, s’ha de pausar el SFX i la música si estava sonant. - 
Han d’aparèixer Sliders que permetin modificar el valor dels VCAs 
corresponents (General, SFX i Music).
